# Bolão B55 - Copa do Mundo 2026 ⚽🏆

Página interativa de bolão (pool de apostas) para a Copa do Mundo 2026.

## 🎯 Funcionalidades

- ⚽ **Apostas**: Faça palpites no placar dos 2 jogos do Brasil
- 🏆 **Ranking**: Veja o ranking em tempo real com a pontuação
- 📋 **Regras Claras**: Sistema de pontuação transparente
- ⚙️ **Painel Admin**: Insira os resultados dos jogos
- 💾 **Dados Compartilhados**: Todos que acessam o link veem as mesmas apostas
- 🔒 **Seguro**: As apostas não podem ser alteradas após criação

## 🎮 Como Usar

1. Digite seu nome
2. Escolha o placar dos 2 jogos:
   - 🇧🇷 Brasil vs 🇭🇹 Haiti (19/06)
   - 🇧🇷 Brasil vs 🏴󠁧󠁢󠁳󠁣󠁴󠁿 Escócia (24/06)
3. Clique em "Colocar Aposta"
4. Aguarde os resultados
5. Veja seu ranking!

## 🎁 Prêmio

**🏆 Mochila GoCase Personalizada B55** para o vencedor!

## 📊 Regras de Pontuação

| Acerto | Pontos |
|--------|--------|
| Resultado Exato | 10 |
| Vencedor + 1 Placar | 5 |
| Apenas Vencedor | 3 |
| Apenas Um Placar | 1 |

## 🚀 Deploy no GitHub Pages

1. Crie um novo repositório no GitHub: `bolao-b55`
2. Clone para seu computador:
   ```bash
   git clone https://github.com/SEUUSER/bolao-b55.git
   ```
3. Copie os arquivos para dentro da pasta
4. Faça o commit:
   ```bash
   git add .
   git commit -m "Bolão B55 Copa do Mundo 2026"
   git push origin main
   ```
5. Ative GitHub Pages:
   - Vá em Settings > Pages
   - Selecione "main" como branch
   - Escolha "/" (raiz) como pasta
6. Pronto! Sua página estará em: `https://SEUUSER.github.io/bolao-b55/`

## 📱 Compatibilidade

- ✅ Desktop (Chrome, Firefox, Safari, Edge)
- ✅ Mobile (iOS, Android)
- ✅ Tablet

## 💡 Notas

- Os dados são salvos em `localStorage` do navegador
- Todos que acessam o mesmo link compartilham os mesmos dados
- Os dados persistem mesmo fechando o navegador
- Não é necessário banco de dados ou backend
